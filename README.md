## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone [https://github.com/phoenixdev100/portfolio-1.git]
   cd portfolio-1
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the server**
   ```bash
   npm start
   ```

4. **Open your browser**
   ```
   Navigate to http://localhost:3000
   ```
